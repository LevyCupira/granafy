# LGPD - Incidentes e Atendimento ao Titular

Ultima atualizacao: 31/05/2026

## 1. Objetivo

Estabelecer um roteiro minimo para dois temas que nao podem depender de improviso:

- atendimento a direitos do titular
- resposta a incidente de seguranca com dados pessoais

## 2. Direitos do titular - rito basico

Quando houver pedido relacionado a dados pessoais, seguir este fluxo:

1. registrar a solicitacao
2. identificar quem pediu e com qual legitimidade
3. identificar qual base, cliente ou conta esta envolvida
4. localizar os dados
5. responder dentro de prazo razoavel, com rastreabilidade

Pedidos que o processo deve contemplar:

- confirmacao da existencia de tratamento
- acesso aos dados
- correcao de dados incompletos, inexatos ou desatualizados
- informacao sobre compartilhamento
- eliminacao, bloqueio ou anonimizacao quando juridicamente cabivel
- revogacao de consentimento, quando a base legal efetivamente for consentimento

## 3. Incidente de seguranca - rito minimo

Ao identificar suspeita de incidente:

1. conter o acesso ou a falha
2. preservar evidencias minimas
3. registrar data, hora, sistema e impacto percebido
4. identificar quais dados podem ter sido afetados
5. avaliar se ha risco ou dano relevante aos titulares
6. decidir se ha necessidade de comunicacao

## 4. Registro minimo de incidente

Cada incidente deve registrar:

- data da deteccao
- origem
- modulo afetado
- tipo de dado afetado
- quantidade estimada de titulares
- medidas de contencao
- medidas de correcao
- necessidade ou nao de notificacao
- responsavel pela decisao

## 5. Medidas imediatas recomendadas para o Granafy

- revisar acessos compartilhados por cliente
- revisar perfis ativos
- revisar logs e trilhas de auditoria disponiveis
- evitar armazenamento desnecessario em campos livres
- reforcar orientacao sobre backup local

## 6. Proximos passos tecnicos

- criar area administrativa simples para registrar incidentes
- criar exportacao orientada ao titular
- criar fluxo de revogacao de acesso compartilhado com trilha de auditoria clara
- criar rotina de revisao periodica de perfis e acessos
