## Testes E2E

- Instalar dependencias: `npm install`
- Instalar navegador do Playwright: `npx playwright install chromium`
- Rodar testes: `npm run test:e2e`
- Rodar com navegador visivel: `npm run test:e2e:headed`
- Abrir modo interativo: `npm run test:e2e:ui`
- Conferencia visual assistida: `npm run visual:check`
